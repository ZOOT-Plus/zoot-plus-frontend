# 验证用例（关卡选择 Dialog）

- 打开/关闭：点击按钮打开；Esc/取消按钮关闭。
- 选择：在对话框内完成“游戏/分类/关卡”筛选；鼠标选择关卡或仅选择筛选关键字后点击“搜索”。
- 边界：
  - 空数据：显示“无匹配结果”。
  - 长列表：列表可滚动（最大 50vh）。
  - 禁用态：按钮禁用，不可打开。
- 一致性：
  - 打开后不被任意 Overlay 遮挡。
  - 与“选择密探”风格一致（标题、底部按钮）。
- 集成：
  - 点击“搜索”：优先提交具体关卡，否则提交拼接后的关键字（含“如鸢/代号鸢”时的“通用”扩展）。
  - 点击“重置”：清空关卡与关键字，并关闭对话框。
